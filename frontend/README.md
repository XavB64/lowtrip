# Frontend — Lowtrip

The frontend is a **React + TypeScript** application responsible for:

- Rendering the interactive map
- Handling user inputs (trip simulation, period selection, etc.)
- Displaying charts and insights
- Managing responsive/mobile UI

---

## 🚀 Getting started

### Prerequisites

- Node.js (>= 18 recommended)
- npm

---

### Run the development server

```bash
cd frontend
npm ci
npm run start
```

The app will be available at:

👉 http://localhost:3000

---

## 🎨 UI & UX

The frontend follows a **mobile-first approach** with:

- Responsive layout (mobile / desktop)
- Scroll-based navigation between main sections (map & chart)
- Interactive components (modals, selectors, tooltips)

---

## 🔧 Available scripts

```bash
npm run start     # Start dev server
npm run build     # Build for production
npm run test      # Run tests
```
